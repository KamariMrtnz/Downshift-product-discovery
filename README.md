# Downshift Product Discovery

A lightweight product discovery experience built with HTML, CSS, and vanilla JavaScript as part of Downshift's Founding Engineer take-home assessment.

## Features

- Instant search across product title, brand, category, description, and tags
- Category filtering
- Product sorting
- Client-side pagination
- Adjustable products per page
- Responsive layout
- Graceful image fallbacks
- Dynamic rendering from a 4,000+ product JSON dataset

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Start a local development server (Live Server recommended).
4. Open `index.html` in your browser.

## Design Decisions

The objective was to build a focused product discovery experience rather than a complete storefront.

Given the one-hour time constraint, I prioritized the core user experience by focusing on:

- Fast, responsive search
- Simple category filtering
- Predictable sorting
- Clean, intuitive browsing

## Future Improvements

With additional time, I would explore:

- Typo tolerance
- Synonym matching
- Improved relevance scoring
- Personalized recommendations
- Product bundling and complementary product suggestions
