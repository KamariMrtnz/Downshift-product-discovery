# Downshift Product Discovery

A lightweight product discovery experience built with HTML, CSS, and vanilla JavaScript for Downshift's Founding Engineer take-home exercise.

## Features

- Search across title, brand, category, description, and tags
- Category filtering
- Product sorting
- Client-side pagination
- Adjustable products per page
- Responsive layout
- Graceful image fallbacks
- Dynamic rendering from a 4,000+ product JSON dataset

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

1. Clone the repository.
2. Open the project in VS Code.
3. Run a local server (Live Server recommended).
4. Open `index.html`.
